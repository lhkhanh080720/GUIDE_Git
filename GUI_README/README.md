# FONTS + TẠO BẢNG
Canh lề bằng dấu ":"
|Font|Cách tạo|Kết quả|
|---|------------------------|----------|
|Đậm|2 dấu * trc và sau từ đó|**oke la**|
|Nghiêng|1 dấu * trc và sau từ đó|*oke la*|
|Đậm + nghiêng|3 dấu * trc và sau từ đó|***oke la***|
|Gạch bỏ|2 dấu ~ trc và sau từ đó|~~delete~~|
|Giữ lại * _|dùng \ |\*  \_|
